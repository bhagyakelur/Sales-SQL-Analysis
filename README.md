Sales Data Analysis using SQL

Overview:
This project analyzes a sales dataset using SQL to extract meaningful business insights such as top-selling products, revenue trends, and customer purchasing behavior.

 Data Cleaning:
* Converted data types (TEXT → INT/FLOAT)
* Standardized date formats
* Extracted city information from address
* Removed duplicate records using window functions

Key Analysis Performed:
* Top-selling products by quantity and revenue
* Revenue distribution across cities
* Peak sales hours and time-based trends
* Monthly revenue trends
* Market basket analysis (products frequently bought together)

Key Insights:
* Evening hours show higher purchasing activity
* Certain products contribute significantly to total revenue
* Sales vary across cities, indicating regional demand differences
* Some product combinations are frequently purchased together

Tools Used:
* SQL (SQLite)

This project demonstrates how SQL can be used for real-world data cleaning, transformation, and business analysis.
